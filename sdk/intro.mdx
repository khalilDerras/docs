# SDK Overview

Welcome to the SDK documentation for the Billing Service. This section provides all the necessary information to get started with our SDK, including setup, core components, and custom hooks.

Our SDK is designed to simplify the integration of billing functionalities into your application, offering a flexible and customizable solution for managing subscriptions, payments, and entitlements.

## Features

- Easy integration with your application
- Comprehensive subscription management
- Customizable billing plans
- Entitlement management
- Built-in components and hooks
